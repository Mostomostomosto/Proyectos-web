Un poco de HTML

Encabezados

En HTML existen 6 niveles de encabezados.

Por ejemplo, con la etiqueta <h1>Título principal</h2> obtendremos un encabezado de nivel 1

Listas

Ejemplo de una lista ordenada

		
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
		
	
Ejemplo de una lista desordenada

	
		
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
